| **Estimated runtime** | ~3 seconds |

---

## Sampling Rate

- **After every task commit:** Run quick check (file exists + frontmatter present for the SKILL.md just touched)
- **After every plan wave:** Run full `phase6-structure-check.sh` (~17 assertions across all 3 platforms)
- **Before `/gsd-verify-work`:** Full suite must exit 0 with all assertions PASS
- **Max feedback latency:** 5 seconds
